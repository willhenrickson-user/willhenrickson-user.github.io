Projects:

Here are two examples of projects I have completed during my time at CU Boulder that I believe are indicative of my technical skills and interests.

Project 1: Using Spotify Data to Determine what Makes a Song Danceable.

This project involved pulling 100 songs from 6 different musical genres through the Spotify API and using the built in analysis features to figure out how Spotify determines the danceability of a song using data analysis.


Project 2: Using Data Visualizations to Compare Drunk Driving Arrest to Liquor Lisence Holders.

Completed as a group, this project involved retrieving both DUI arrest incidents and liquor liscnese owners in both Boulder and Denver to determine if there was a relationship between alcohol vendors and drunk driving incidents.
